🚀 ✅ What You Need for Chrome (Flutter Web)

1. Flutter SDK (same as before)

👉 Already required


---

2. Enable Web Support in Flutter SDK

Run this once:

flutter config --enable-web


---

3. Install Google Chrome

👉 Flutter uses Chrome to run your app

Check:

flutter devices

You should see:

Chrome (web)


---

4. Verify Setup (VERY IMPORTANT)

Run:

flutter doctor

Look for:

[✓] Chrome - develop for the web

If this line is missing → Chrome not detected ❌


---

▶️ How You Will Run in Exam (SUPER SIMPLE)

flutter create myapp
cd myapp
flutter run -d chrome

👉 This will:

Start a local server (localhost)

Open app in Chrome automatically


Example:

http://localhost:12345


---

🌐 What “localhost” Means (Simple)

Your app runs on your own laptop

Chrome opens it like a website

No internet required
